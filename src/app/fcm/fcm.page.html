Firebase Cloud Messaging Topic-based Push Notifications

This page demonstrates topic-based push notifications using Firebase Cloud Messaging (FCM). When the app is opened, it will display the notification in the foreground. When closed, it will send the notification to the native device (or browser via the service worker). First, we need to get permission to send.
